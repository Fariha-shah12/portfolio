Flood Vulnerability in Asian Countries

Floods are one of the most frequent and devastating natural disasters in Asia. This project aims to predict the number of flood disasters using machine learning regressors, forecast future flood trends using SARIMA (Seasonal ARIMA), and analyze key disaster-related indicators and their relationships to flood frequency.

Bird Species Classification using Convolutional Neural Networks

In this project, a CNN-based neural network is developed to classify bird species in Seattle using spectrograms generated from bird call recordings. The model was trained for binary (e.g., American Crow vs. House Sparrow) and multi-class classification (12 species total), and evaluated using external test clips. The dataset was sourced from Xeno-canto, and spectrograms served as input to the model for effective sound recognition.

Health Behavior and Diabetes Prediction using (SVM)

This project explores the relationship between health behaviors, demographic factors, and the risk of diabetes using Support Vector Machine (SVM) models. The analysis is based on data from the National Health Interview Survey (NHIS). We evaluated the performance of Linear, Polynomial, and RBF kernel SVMs to classify individuals as diabetic or non-diabetic.
